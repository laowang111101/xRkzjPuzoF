# 基于Springboot的校园工作室权限后台管理系统 java174

## 项目概述

本项目是一款基于Springboot框架的校园工作室权限后台管理系统，主要用于管理校园工作室的个人和团队信息，以及系统权限的分配。系统设计简洁，功能模块清晰，便于工作室成员进行个人信息的录入和管理，以及工作室整体的运营与维护。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js + Element UI
- 数据库：MySQL

## 功能模块

### 个人管理模块

- 信息录入
- 获奖证书上传
- 个人竞赛记录
- 个人心得分享

### 工作室管理模块

- 专业管理
- 年级管理
- 竞赛管理
- 证书管理
- 成员管理

### 系统管理模块

- 菜单管理
- 角色管理

### 系统主页

- 大数据看板：图形化界面展示数据

## 系统角色

- 普通用户：可以查看和编辑个人信息，参与竞赛等
- 管理员：负责工作室的日常管理，包括成员管理、证书管理等
- 超级管理员：拥有最高权限，可以管理系统菜单和角色

## 运行环境

- 服务器：推荐使用Linux服务器
- 数据库：MySQL 5.7+
- 后端：JDK 1.8+
- 前端：现代浏览器

## 部署步骤

1. 准备运行环境（服务器、数据库、JDK等）
2. 部署后端服务（Spring Boot应用）
3. 部署前端服务（Vue.js应用）
4. 启动应用并访问
